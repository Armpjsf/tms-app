คู่มือการใช้งาน LINE Bot

ระบบจัดการขนส่ง Elite TMS & ePOD

1. การลงทะเบียนผูกบัญชี (Registration)

เพื่อให้ LINE Bot รู้จักตัวตนและสิทธิ์การใช้งานของคุณ กรุณาทำตามขั้นตอนดังนี้:

  1. เพิ่มเพื่อน (Add Friend): สแกน QR Code หรือแอดไอดี LINE Bot ของบริษัท
  2. กดเมนู "ลงทะเบียน": ที่เมนูด้านล่างหน้าจอ (Rich Menu) หรือพิมพ์คำว่า ลงทะเบียน
  3. กรอกข้อมูลระบุตัวตน: ระบบจะส่งลิงก์ให้คุณกรอก "เบอร์โทรศัพท์" หรือ "รหัสพนักงาน (Username)" ที่ลงทะเบียนไว้ในระบบ TMS
  4. ยืนยันสำเร็จ: เมื่อระบบตรวจสอบข้อมูลตรงกัน คุณจะได้รับข้อความยืนยันการเชื่อมต่อ และสามารถเริ่มใช้คำสั่งต่างๆ ตามสิทธิ์ของคุณได้ทันที
2. ชุดคำสั่งแยกตามสิทธิ์ผู้ใช้งาน (Commands)

คุณสามารถพิมพ์ข้อความเหล่านี้ลงในช่องแชทเพื่อให้บอทดึงข้อมูลให้ทันที (ข้อมูลที่แสดงจะถูกกรองตามสิทธิ์ของคุณ)

👑 สำหรับ Super Admin (ผู้บริหาร)

เข้าถึงข้อมูลได้ทุกสาขาและภาพรวมบริษัท

  • พิมพ์ สรุปยอดวันนี้ ดูจำนวนงานทั้งหมด, งานที่สำเร็จ, งานค้าง และเป้าหมายรายวัน
  • พิมพ์ สรุปรายได้เดือนนี้ ดูยอดรายรับรวม, ค่าใช้จ่าย, และกำไร (Profit) แบบเรียลไทม์
  • พิมพ์ สถานะรถทั้งหมด ดูว่ารถคันไหนวิ่งอยู่ (Online) คันไหนจอด หรือคันไหนขาดการติดต่อ
  • พิมพ์ มีรถคันไหนซ่อมอยู่บ้าง ตรวจสอบรถที่แจ้งซ่อมหรือถึงกำหนดเข้าศูนย์บริการ
  • พิมพ์ แจ้งเตือน SOS ตรวจสอบการแจ้งเหตุฉุกเฉินจากคนขับรถในวันนี้

💻 สำหรับ Admin (พนักงานสาขา)

เข้าถึงข้อมูลเฉพาะสาขาที่ตนเองรับผิดชอบ

  • พิมพ์ งานของสาขาวันนี้ ดูสถานะงานขนส่งเฉพาะของสาขาตัวเอง
  • พิมพ์ ตามงาน [รหัสงาน] เช่น 'ตามงาน JOB-12345' เพื่อดูพิกัดล่าสุดและสถานะงานนั้นๆ
  • พิมพ์ มีรถคันไหนว่างบ้าง เช็ครายชื่อคนขับในสาขาที่ยังไม่มีงานติดตัว
  • พิมพ์ ดูพิกัด [ทะเบียนรถ] ตรวจสอบตำแหน่งปัจจุบันของรถคันที่ต้องการ

🏢 สำหรับ ลูกค้า (Customer)

เข้าถึงข้อมูลได้เฉพาะงานขนส่งของบริษัทตนเองเท่านั้น

  • พิมพ์ เช็คสถานะสินค้า ดูรายการสินค้าที่กำลังมาส่งในวันนี้ พร้อมประเมินเวลาถึง (ETA)
  • พิมพ์ ติดตามงาน [รหัสงาน] ดูสถานะและตำแหน่งรถของงานที่ระบุ
  • พิมพ์ สรุปค่าใช้จ่ายเดือนนี้ ดูยอดรวมค่าขนส่งที่ต้องชำระในรอบบิลปัจจุบัน
  • พิมพ์ ขอดูหลักฐาน [รหัสงาน] เรียกดูรูปถ่ายสินค้าและลายเซ็น (ePOD) ของงานที่ส่งสำเร็จแล้ว

🚚 สำหรับ คนขับรถ (Driver)

จัดการงานของตนเองและสื่อสารกับแอดมิน

  • พิมพ์ งานของฉันวันนี้ เรียกดูรายการจุดรับ-ส่งสินค้าที่ต้องวิ่งในวันนี้
  • พิมพ์ ค่าเที่ยวเดือนนี้ ตรวจสอบยอดเงินรวมและค่ารอบที่สะสมในเดือนปัจจุบัน (หากแอดมินเปิดให้ดูได้)
  • พิมพ์ ส่งตำแหน่ง (การแชร์ Location ผ่าน LINE) เพื่ออัปเดตพิกัดให้แอดมินทราบทันที
  • พิมพ์ ติดต่อแอดมิน ขอความช่วยเหลือหรือแจ้งปัญหาให้แอดมินสาขาทราบ
